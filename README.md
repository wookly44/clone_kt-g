## 🌱 배운 점 / 느낀 점

- 말줄임표(`text-overflow: ellipsis`)를 사용해 텍스트 길이 처리 방법을 익힘
- CSS를 `reset.css`, `common.css`, `style.css`로 분리하여 구조적으로 관리하는 방식을 적용
- 다양한 CSS 애니메이션을 적용하여 인터랙션 구현

## ⚠ 아쉬운 점 / 개선하고 싶은 부분
